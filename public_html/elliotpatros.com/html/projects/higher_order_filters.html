higher order filter project

The higher order filter project is a collection of resonant filter objects for Miller Puckette's Pure Data and Max.

The goal of the HOFP is to provide musicians and sound designers with popular resonant filters that aren't easily found in patching environments. HOF objects are super efficient without sacrificing accuracy, and have intuitive user interfaces.

All higher order filter objects come with in-app help patches.

installation

  • Download and expand the appropriate .zip for your computer.
  • (Optional) Copy the folder to any folder you prefer.
  • Add the directory of hofp-* to Pd's search paths (File >>> Preferences >>> Path...).
  • click Apply, then OK, then restart Pd.

object overview

second-order filters: highpass~, lowpass~, bandpass~, notch~, allpass~.
Biquad filters with variable frequency and Q.
equalizer filters: highshelf~, lowshelf~, peak~.
Biquad filters with variable frequency, Q, and gain.
nth order filters: fir~
Finite impulse response filter. Set coefficients dynamically by referencing table or array objects.
