tweaq (This Will Edit Audio Quickly)

tweaq is a lightweight desktop application for batch processing audio files.

Built for folks who work with large collections of audio files, it makes repetitive tweaks super fast. tweaq is an extensible, open-source software package that specializes in simple but common dsp algorithms.

To use tweaq, drag and drop sound files or folders into the main window. Pick a process from the dropdown menu, and click the big button. Processed files will appear in a destination of your choice. tweaq will not overwrite files, so you can experiment freely.

tweaq uses the Qt application framework and Erik de Castro Lopo's libsndfile.
